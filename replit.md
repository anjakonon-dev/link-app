# Линк — веб-чат с аудио

## Как запустить

1. Нажмите Run в Replit
2. Перейдите по ссылке Replit
3. Введите email (отображается имя до @)
4. Используйте чат и аудиозвонки

## Структура проекта

- server.js — сервер Node.js + Socket.IO
- package.json — зависимости
- public/
  - index.html — интерфейс
  - client.js — логика чата
  - style.css — стили
